# Questly

Questly is a browser-based **Question Generator** designed to help students study more effectively. Upload your notes or study material, and Questly will automatically generate intelligent, topic-based questions — making revision faster and more structured.

## Features

- **File Upload** — Supports `.pdf` and `.txt` file formats for direct upload in the browser
- **Automatic Question Generation** — Extracts content from uploaded material and generates relevant study questions
- **Dark / Light Mode** — Toggle between themes based on preference
- **Minimal Interface** — Clean, distraction-free design focused on usability

## Built With

- **HTML5** — Page structure and layout
- **CSS3** — Styling, gradients, and theme switching
- **JavaScript** — File handling and question generation logic
- **PDF.js** — Client-side text extraction from PDF files
